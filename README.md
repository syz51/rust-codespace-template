# Codespace VSCode Template for Rust
This is a bare minimum starting template for running a Rust dev environment on codespace.

Template details:
- Debian 13
- Rustup
- Cargo
- Rustfmt
- Clippy
- Zsh/Oh My Zsh

VS Code extentions included:
- rust-analyzer
- lldb
- docker
- crates
- even-better-toml

Contributions welcome
